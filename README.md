## *Chạy phần mềm*

Yêu cầu Visual Studio 2022 và .NET 8.0 

## *Các thuật toán*

- DFS
- BFS
- Kruskal
- Prim
- Dijkstra

## *Features*
- Fullscreen mode
- Lưu/Load đồ thị
- Zoom in/out map
- Di chuyển map
- Ma trận kề/trọng số
